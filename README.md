## ⚙️ Local Setup

1️⃣ **Clone the repository:**

```bash
git clone https://github.com/samyuktha2005/myrepo.git
cd myrepo/my-app
````

2️⃣ **Install dependencies:**

```bash
npm install
```

3️⃣ **Run the development server:**

```bash
npm start
```

4️⃣ Open your browser and navigate to:

```
http://localhost:3000
```
